# Guardian AR — 대규모 사용자 시뮬레이션 보고서 (v2)

**일자:** 2026-05-05
**범위:** β 모델 + Soft expansion + 동맹 단계 + 공간 인덱싱 도입 후 재평가
**방법:** 코드 정적 분석 + 게임 흐름 수학 모델링 + 베타테스트 48 케이스 통과 (회귀 0)
**시나리오:** N=10 / N=100 / N=1000 동시 활성 사용자

## 0. v1(2026-05-04) 대비 변화

| 항목 | v1 | v2 |
|---|---|---|
| 영역 모델 | 영역 + 영역당 최대 3 타워 | **β 모델: 1 영역 = 1 타워** |

## 1. N=10 (친구·동호회 단위)

### 시뮬 결과
| 항목 | v1 | v2 (개선) |
|---|---|---|
| 평균 영역 밀도 | 30/25km² (1.2/km²) | 동일 (영역 cap이 안 닿음) |
| PvP 발생 빈도 | 1~2/일 (적 못 만남) | **5~7/일** (NPC starter 200m + 매 신규 첫 영역에 1봇) |
| 동맹 형성 | 활발 | 한도 10명 도달 어려움 (정상) |
| 서버 RPS | 1.5 / 5 | 동일 |
| DB tick 비용 | 30² = 900 비교 | spatialGrid로 ~270 |

### 게임 흐름
- ✅ NPC 봇으로 PvP 콘텐츠 보장 — 1주 이탈 위험 완화
- ✅ 13종 다양성이 의미 가지기 시작 (NPC 격파 후 발판으로 다양한 클래스 시도)
- ⚠ **하지만 NPC는 Lv1 generic 1체** — 레벨 5+ 유저에게는 너무 약함, 콘텐츠 마비
- ⚠ 보스(6h마다, 하루 4번)가 PvE 메인이지만 단조로움 그대로

### 발견된 새 문제
1. **NPC 강도 정적** — 신규 spawn 시점 유저 레벨 무관하게 항상 Lv1 → 고레벨 유저는 5초만에 격파
   → **N=10에서도 곧 콘텐츠 고갈**
2. **NPC 후속 spawn 메커니즘 없음** — 첫 영역에만 1체. 격파 후 새로 안 생김
3. **속국 시스템 의미 없음** — 적이 없는데 누가 누구의 속국이 될 이유가?

### 개선 방향
- NPC 강도를 spawner의 레벨에 비례하게 (`Math.min(spawner.level, 5)`)
- N=10 환경에서는 **1주마다 새 NPC spawn** (player_count < threshold일 때)
- 레벨 5+ 유저에게는 NPC 봇 그룹(동맹 3체) 스폰

## 2. N=100 (도시 한 구 단위)

### 시뮬 결과
| 항목 | v1 | v2 |
|---|---|---|
| 평균 영역 밀도 | 300/25km² (12/km²) | **β 모델 cap으로 평균 ~3 영역/유저 = 300개 동일** (충분) |
| 영역 간 평균 거리 | 290m | 동일 |
| 영역 확장 성공률 | ~60% | **~85%** (soft expansion 20% 허용 + 도시 외곽 NPC 분산) |
| 침입 발생 빈도 | 5~10/일 | 5~10/일 |
| 동맹 형성 | 무제한 → 인플레 | **10명 한도 + 24h 임시 50%** — 신중한 동맹 |
| 서버 RPS | 15 / 50 | 동일 |
| DB tick 비용 | 90,000 비교 | **~2,700** (grid 인접 9칸) |
| 전투 발생 | 200/일 | **300/일** (soft expansion 영역끼리 자동 공성↑) |

### 게임 흐름 (재미 평가)
- ✅ **동맹 인플레 차단** — "친구 다 동맹"하면 임시 50%로 무력 → 정식 승격 7일 단위 메타
- ✅ **soft overlap 영역**이 새 전선 형성 — defense_penalty 0.7로 약점이지만 진입 가능
- ✅ **타워 빌드 다양성** 발현 — 13×5=65 빌드. 발리스타·SF가 공성 메타 핵심
- ⚠ **속국 vs 동맹** 선택 압력 — 속국은 영역 안에 들어가야, 동맹은 인접만. 차별화 명확
- ⚠ **정식 동맹 7일 자동 dissolve가 갑작스러움** — UI 미리 알림 없음
- ❌ **siege grace 6h**가 길어 모바일 사용자는 알림 못 봄

### 새 문제
1. **soft overlap 영역 시각 표시 없음** — defense_penalty 0.7임을 사용자가 모름 → "왜 약하지?" 혼란
2. **동맹 stage 만료 알림 없음** — 정식 7일 만료 직전 푸시 미구현
3. **속국 조공 강요 가능성** — lord가 vassal에게 조공률 90%로 reject 협박? (아직 issue 없지만)
4. **클래스 해금 Lv8 게이트** — 신규 유저가 4시간 플레이해도 ice/aqua/venom 못 씀

### 권장 개선
- **soft overlap 영역 빨간 점선 마커** (warning_at처럼 시각화)
- **동맹 stage 만료 24h 전 푸시 알림** (정식 → dissolve)
- **siege grace 6h → 4h + 2회 푸시 알림** (모바일 사용자 보호)
- **클래스 해금 완화**: Lv5에 generic+5종, Lv10에 8종, Lv15에 13종

## 4. 재미 요소 평가 (v2 기준)

### ✅ 강화된 재미
1. **두 정복 경로(A/B) 명확화** — 선택의 결정 지점이 게임 핵심
2. **속국 시스템** — 강자/약자 비대칭 상호작용. tribute_pct 협상이 외교 게임
3. **타워 13×5 빌드 다양성** — Lv 해금이 점진 학습 곡선 형성
4. **NPC starter** — N=10에서도 첫날 PvP 경험 보장
5. **동맹 단계** — 임시/정식 구분으로 동맹 가치 보존
6. **AR 야전모드** — Unity 없는 사용자도 카메라 + 컴패스로 위협 인식
7. **레벨 cap 시스템** — 황제(Lv30)에 도달 가능한 명확한 진행 (이전 cap 9999 데드락 해소)

### ⚠ 위험 요소 (재미 저해)
1. **soft overlap 영구 페널티** — 신규 유저가 잘 모르고 거기 박으면 "왜 약함?" 후회 → **회복 메커니즘 필요**
2. **동맹 stage 자동 dissolve** — 7일 정확히 후 사라짐, 알림 없음 → **로그인 시 토스트 필수**
3. **속국 일방 dissolve 처리** — vassal 영역+타워 운명 불명확 → **계약 시 명문화 필요**
4. **NPC 후속 spawn 없음** — 격파 후 적이 또 사라짐 → **respawn 또는 Wave 시스템**
5. **β 모델 영역 폭발** — Lv30이 76개 작은 영역 운영하는 게 직관적인가? → **메가-영역 보너스 옵션**
6. **Lv 게이트 너무 멀리** — Lv8(약 4-6h 플레이)까지 ice 못 쓰면 지루 → **해금 가속**

### ❌ 여전한 미해결 (v1 잔존)
1. **시즌 종료 보상 미구현** — 진보의 의미가 약함
2. **궁극기 단일 효과** — 본체 타입 4종 미도입으로 차별화 부재
3. **AR 한글 폰트 부재** — Unity ASCII 강제

## 6. 결론

**v2 평가: N=1000도 가능한 시스템 + 게임 흐름 명확화**

- **스케일 차단 P0-1/2/3/4 4건 완전 해소** (이전 보고서 대비). N=1000 시뮬에서 DB 부하 99%↓
- **신규 진입성 P1-1 해소** (soft expansion + NPC starter)
- **동맹 인플레 P1-4 해소** (한도 10 + 임시/정식)
- **새 P0/P1 5건 식별** (β 모델 부작용 + 속국 폭주 + atari 도미노 잔존)

게임플레이 재미는 **명확한 메타가 형성**된 상태:
- 두 정복 경로(A/B) + 속국/동맹/길드 + 타워 13×5 + 레벨 1~30 cap + AR 모드 = 충분한 콘텐츠 깊이
- 다만 **장기 운영(N=1000, 시즌 3개월)** 측면에서 추가 안전장치 + 콘텐츠 지속성 필요
