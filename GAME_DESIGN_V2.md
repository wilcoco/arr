# Guardian AR - 확장 기획서 v2

## 세계관

### 배경 스토리
2030년, 전 세계에 "균열(Rift)"이 발생하며 차원의 틈에서 에테르(Aether) 에너지가 유출되기 시작했다. 이 에너지는 특정 위치에 "결정점(Node)"을 형성하며, 인류는 이 에너지를 활용해 수호신(Guardian)을 소환할 수 있게 되었다.

수호신은 소환자의 의지와 결합하여 형태를 갖추며, 에테르 에너지를 흡수해 성장한다. 각 지역의 결정점을 차지하기 위한 수호신들의 영역 전쟁이 시작되었다.

### 핵심 자원

| 자원 | 아이콘 | 획득 방법 | 용도 |
|------|--------|-----------|------|
| **에테르(Aether)** | 💎 | 생산형 수호신, 전투 승리 | 영역 확장, 수호신 강화 |
| **코어(Core)** | 🔮 | 고정 수호신 파괴, 일일 퀘스트 | 본체 능력치 영구 강화 |
| **결정편(Shard)** | ✨ | 영역 유지 보상, 동맹 활동 | 궁극기 충전, 특수 스킬 |

### 능력치 상세

| 능력치 | 약어 | 효과 |
|--------|------|------|
| 공격력 | ATK | 전투 데미지 기본값 |
| 방어력 | DEF | 받는 데미지 감소 |
| 체력 | HP | 전투 지속력, 0이 되면 패배 |
| 흡수율 | ABS | 승리 시 상대 스탯 흡수 % |
| **생산력** | PRD | 시간당 에테르 생산량 |
| 속도 | SPD | 선제 공격 확률, 회피율 |
| 사거리 | RNG | 원격 지원 거리 (m) |
| 영역력 | TER | 영역 확장 효율 (비용 감소) |

### 본체 강화 시스템

**코어(🔮) 사용:**
```
1 코어 = 선택한 능력치 +1 (영구)
강화 비용: 현재 능력치 × 2 코어
예) ATK 15 → 16 강화 시 30 코어 필요
```

**에테르(💎) 사용:**
```
임시 버프: 10 에테르 = 모든 스탯 +10% (1시간)
회복: 5 에테르 = HP 완전 회복
```

## 영역 시스템 확장

### 영역 확장 비용

영역 크기에 따른 에테르 비용:

| 반경 | 에테르 비용 | 유지 비용/시간 |
|------|-------------|----------------|
| 50m | 10 💎 | 1 💎 |
| 100m | 25 💎 | 3 💎 |
| 200m | 60 💎 | 8 💎 |
| 300m | 120 💎 | 15 💎 |
| 500m | 250 💎 | 30 💎 |

**TER(영역력) 효과:**
```
실제 비용 = 기본 비용 × (100 / (100 + TER))
예) TER 20이면 비용 17% 감소
```

### 영역 크기 vs 방어력 트레이드오프

**핵심 규칙: 넓은 영역 = 약한 방어**

```
영역 방어 계수 = 100 / (반경(m) / 50)

예시:
- 50m 영역: 방어 계수 100% (풀 방어력)
- 100m 영역: 방어 계수 50%
- 200m 영역: 방어 계수 25%
- 500m 영역: 방어 계수 10%
```

**전투 시 적용:**
```
실제 방어력 = (본체 DEF + 고정수호신 DEF) × 영역 방어 계수
```

**전략적 선택:**
- 작은 영역 여러 개 = 강한 방어, 관리 복잡
- 큰 영역 하나 = 약한 방어, 관리 쉬움, 생산 효율

### 영역 유지 실패

에테르 부족 시:
1. **경고 단계** (12시간): 생산량 50% 감소
2. **약화 단계** (24시간): 영역 방어력 0, 누구나 점령 가능
3. **소멸** (48시간): 영역 자동 삭제, 고정 수호신 파괴

## 생산 시스템

### 에테르 생산

**본체 수호신 생산:**
```
시간당 생산량 = PRD × 0.5
예) PRD 25 = 시간당 12.5 에테르
```

**생산형 고정 수호신:**
```
시간당 생산량 = 배치된 PRD × 1.0
예) PRD 10 배치 = 시간당 10 에테르
```

**영역 보너스:**
```
영역 내 생산 보너스 = +20%
동맹 영역 인접 보너스 = +10%
```

### 생산물 사용처

| 용도 | 에테르 비용 | 효과 |
|------|-------------|------|
| 영역 확장 | 10~250 | 새 영역 생성 |
| 영역 유지 | 1~30/시간 | 영역 유지 |
| 고정 수호신 배치 | 20 | 방어/생산 수호신 배치 |
| 본체 임시 강화 | 10 | 1시간 스탯 +10% |
| 본체 회복 | 5 | HP 완전 회복 |
| 원격 지원 | 15 | 동맹 전투에 원격 참여 |
| 영역 이전 | 50 | 영역 위치 변경 |

### 자동 생산 시스템

```javascript
// 서버에서 1시간마다 실행
calculateProduction(userId) {
  본체생산 = guardian.prd × 0.5
  고정생산 = sum(fixedGuardians.prd) × 1.0
  영역보너스 = (본체생산 + 고정생산) × 0.2
  
  총생산 = 본체생산 + 고정생산 + 영역보너스
  유지비용 = sum(territories.maintenanceCost)
  
  순수익 = 총생산 - 유지비용
  user.aether += 순수익
}
```

| 타입 | 아이콘 | 역할 | 배치 비용 |
|------|--------|------|-----------|
| **방어형** | 🛡️ | 영역 방어력 증가 | 20 💎 |
| **생산형** | ⚙️ | 에테르 자동 생산 | 30 💎 |
| **정찰형** | 👁️ | 침입자 조기 경보, 위치 추적 | 25 💎 |

### 능력치 분배

본체에서 고정 수호신으로 능력치 분배:

```
분배 가능량 = 본체 해당 스탯의 50%까지
분배 시 본체 스탯 감소
회수 시 본체 스탯 복구 (10% 손실)

예) 본체 DEF 20에서 DEF 10을 방어형에 분배
    → 본체 DEF 10, 방어형 DEF 10
    회수 시 → 본체 DEF 19 (1 손실)
```

### 고정 수호신 성장

전투에서 방어 성공 시:
```
경험치 획득 = 공격자 총 스탯의 5%
레벨업 시 배치된 스탯 +5%
```

## 전투 시스템 확장

### 전투력 계산 (개선)

**공격자:**
```
공격력 = ATK × (1 + SPD/100) × 랜덤(0.85~1.15)
크리티컬 확률 = SPD / 200 (최대 25%)
크리티컬 데미지 = ×1.5
```

**방어자:**
```
방어력 = (본체 DEF + 고정수호신 DEF + 동맹지원 DEF) × 영역방어계수
회피 확률 = SPD / 300 (최대 20%)
```

### 전투 결과

**승리 시:**
- 상대 스탯 흡수: 패배자 총 스탯 × (승자 ABS / 100)
- 에테르 약탈: 패배자 보유량의 20%
- 코어 획득: 고정 수호신 파괴 시 1~3개

**패배 시:**
- 스탯 손실: 총 스탯의 (상대 ABS / 100)
- 에테르 약탈당함
- 영역 내 전투 시 영역 점령당함

### 원격 지원 (RNG 활용)

동맹이 공격받을 때:
```
지원 가능 거리 = RNG × 10 (m)
지원 효과 = 본체 DEF의 30%를 동맹에게 추가
비용 = 15 에테르
```
